# holiday-extras-developer-api-task-ui
## Installation

Use [HomeBrew](https://brew.sh/) to install [Node.js](https://nodejs.dev/).

```bash
brew install node
```
Install Dependencies.

```bash
npm install
npm i
```

## Lint

```bash
npm run lint
```
## Test

```bash
npm test
npm t
```
